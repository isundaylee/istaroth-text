# Talk Group: NpcGroup - 20523

## Talk 0

志琼: 请问前辈有什么事吗？
旅行者: 关于地图…
志琼: 前辈是说地图吗？没什么好提的啦…在加入冒险家协会之前，我曾经是矿上的制图员。
志琼: 当然啦，我说的不是那些坐办公室画图的制图师…嘿嘿，我还差得远呢。
志琼: 之前受过他们的委托，在翘英庄跑上跑下测绘地形…结果完事建筑计划却取消了…也都是他们一句话的事。
志琼: 他们可以只用委托助手现场勘探，给他们收集地图信息，然后自己才动手工作…但我们这些制图员可不行。
志琼: 所以…身为半个矿工，我也早就习惯在危险的地下通道中穿梭了——工作要求如此，没办法嘛。
志琼: 而且…在异变发生后，层岩巨渊下层其实很…不稳定。
派蒙: 欸…不稳定？很危险吗？
志琼: 嗯，地质结构一直在缓慢地变化，今天的通路，可能到下个月就变成了石壁…就像…唔，我想想怎么说…
志琼: 就像层岩巨渊在随着自己的心思发生变化一样。
派蒙: 哇…这么邪门的吗…
旅行者: 关于冒险家生涯…
志琼: 哈哈，这才哪到哪呀，就「生涯」了！前辈请不要太客气啦，我还只是个新人冒险家而已…跟前辈差得太远啦。
志琼: 虽然之前完成过一些冒险家协会的委托，但我经验尚浅，也并没有独立探索过什么地点，更没有在哪里留下自己的故事…
志琼: 所以我一直在想，层岩巨渊这个地方，说不定能让我留下足迹，让后人记得，我这个人曾经来过。
志琼: 就像大冒险家艾莉丝曾经说的那样，层岩巨渊说不定会是「我的『卢比孔河』」！
旅行者: 「鹿鼻孔河」…是丘丘语吗？
派蒙: 没听说过呢…
志琼: 唔…我也不是很懂…艾莉丝毕竟是穿梭诸世界的大冒险家，或许也是她去过的地方之一吧…？
旅行者: 是「露鼻孔盒」呀！真厉害！
派蒙: 喂，你自己也不知道那是什么东西吧！
志琼: 哈哈，不管怎样，多谢前辈的鼓励！
旅行者: 关于克里托夫…
志琼: 他呀…哼，他这个人我是信不过。我们在矿上和盗宝团不知道周旋过多久了，还能不知道他们的坏心眼？
志琼: 这些人呀…我劝你千万别信他们那套说辞，什么「改过自新」什么「金盆洗手」…
志琼: 如果你认真在听他们说话，他们一定已经把脏手伸到你的口袋里了。
志琼: 假如你相信了他们的鬼话，恐怕岩王爷他老人家也保不住你的钱包咯。
志琼: 哼，我当然不是对他们有偏见…我只是想提醒你小心一点，盗宝团是不可相信的。
旅行者: 你这显然是偏见。
志琼: 也许吧…你作为前辈比我更加见多识广，对盗宝团那些家伙也能更宽容。但我总还要留些心眼盯着他们。
志琼: 除非这家伙能证明，他和那群走私犯同伙没有关系。
旅行者: 再见。
志琼: 前辈再见！
