# Talk Group: NpcGroup - 2009

## Talk 0

德安公: 干什么…烦着呢！

### Talk 0 related talk 0

旅行者: 您是…？
德安公: 哼…现在的年轻人，头发长见识短…
德安公: 十年前，璃月港上至绯云坡，下至各码头，人人都得叫我一声「德安公」…
德安公: 可是如今呢，钱庄也没了，小初也没了…可叹啊可叹，活了这么一辈子，到老了竟遭逢这么多变故…

### Talk 0 related talk 1

旅行者: 需要帮忙吗？
德安公: 哼…帮忙？你帮得了吗！你能让死人复生吗！你能把我产业还给我吗！
德安公: 唉…曾经我也是有自己的钱庄的，明华商行和明华钱庄，二十年前是这条街上最辉煌的门面…
德安公: 可是自从至冬国的那些银行家们仗着愚人众的支持，挤进了璃月的金融市场，我们这些老字号一个个都不好干了…
德安公: 要是我还年轻，说不定还能硬拼一把，拼他个东山再起…
德安公: 可现在我老了，干不动了…
旅行者: 愚人众？
德安公: 怎么，你也听说过他们？
德安公: 这群家伙仗着有至冬国的外交豁免权，在璃月四处探头探脑…可疑得很。
德安公: 而且他们还干涉我们的生意，想方设法要把至冬国的资本打入璃月…
德安公: 也是因为有他们捣鬼，先是钱庄被收购，现在商行的生意也一天不如一天了…

### Talk 0 related talk 2

旅行者: 再见。
德安公: 哼…再见？别见了，别见了…

### Talk 4 related talk 0

旅行者: 您在烦恼什么吗？

德安公: …你该不会是故意来揭我伤疤的吧？走开，走开，别打扰我了！
