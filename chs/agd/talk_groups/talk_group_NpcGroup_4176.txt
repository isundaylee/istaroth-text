### Talk 0 related talk 1

旅行者: 你是？
鲁梅拉: 我是鲁梅拉·阿提夫·艾布·拉希德，须弥的渔师。
鲁梅拉: 别看我很擅长钓鱼，告诉你，我可是出生在沙漠的家伙哦。沙漠虽然有绿洲，但里面几乎没有鱼，我小时候都不知道鱼是什么东西。
鲁梅拉: 直到后面来了港里，才发现我居然这么擅长钓鱼，干这行，我一年能挣人家三年的钱呢！
鲁梅拉: 照这个速度，要不了多久，我就可以把爸爸妈妈接来一起生活了，哈哈，原来这就是我的拉希德。

### Talk 0 related talk 2

旅行者: 拉希德是什么？
鲁梅拉: 拉希德？拉希德就是正确的指引，正确的指引就是拉希德！这是我家乡那边的含义，也是我的名字哦。
鲁梅拉: 我相信万物都有拉希德，比如我的就是钓鱼，鱼的就是被我钓。
鲁梅拉: 嘻嘻，你找到自己的拉希德了吗？

### Talk 0 related talk 3

旅行者: 再见。
鲁梅拉: 再见，祝你早日找到自己的拉希德。
