# Talk Group: NpcGroup - 2005

## Talk 0

李锤子: 过节吗…过节好啊，张灯结彩，比平时热闹多了。
李锤子: 但转念一想，好像和我也没什么关系…

### Talk 0 related talk 0

旅行者: 你是…？
李锤子: …我？你是说我吗…？我就是个苦力，叫我李锤子就好。
李锤子: 这是我自己起的外号，谁叫我一天到晚只能拿着锤子叮叮咣咣呢…
李锤子: 唉，假如什么时候老天开眼，让我也拥有神之眼就好了…
旅行者: 关于工作…
李锤子: 工作…？工作挺好啊，没什么可说的…无非拿着锤子敲敲这里，敲敲那里，时不时被枭老头子骂一顿…
李锤子: 我也老大不小了，反正这样干再久也成不了家，也买不起属于自己的船和宅子，我对未来是没啥指望了…
旅行者: 神之眼？
李锤子: 怎么？你也对神之眼感兴趣吗？
李锤子: 我听说，只要心里对某件东西存在强烈的渴望，「神之眼」就会出现在他身边，供他驱使。
李锤子: 神奇吧？至于我…我就是单纯渴望神之眼。我从小到大，每天做梦都想得到一颗。但是至今还没有那个福气…
李锤子: 假如我有了神之眼，一定也能像故事传说中的那些大侠客一样，擒风使雨、掣雷驱电吧。
李锤子: 唉…神之眼啊…我已经一把岁数了，你在哪儿呢…

### Talk 0 related talk 1

旅行者: 关于岩王帝君
李锤子: …有什么好说的？「请仙典仪」我也在场，可把我吓得不轻…
李锤子: 不过，这事之后，我还以为天衡山要垮了，璃月就这么完了。
李锤子: 但是看到大家仍然像平常一样过日子，也依然对明天抱有希望…我想我也该反思一下自己了。
李锤子: 这样乐观一想的话，岩王爷肯定不是真的离开我们了，你说是吧？
