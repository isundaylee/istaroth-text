### Talk 0 related talk 0

旅行者: 在这里还习惯吗？
基尼奇: 嗯。环境不错，待着也舒适。
阿乔: 我说基尼奇，你的衡量标准会不会低了一点？

基尼奇: #旅行者，感谢你邀请我做客。
基尼奇: 餐风露宿的时候，我只需要一堆篝火，或是一处栖身的山洞。这里提供的不仅是安全，还有别处难以寻得的温馨。
基尼奇: 我应该给你怎样的回报？
旅行者: 不用客气的。
基尼奇: 这不是客气。我不想无偿接受你的好意，这是我的坚持。
基尼奇: 如果你对我有所期望，希望你能直言。我会根据情况考虑如何完成你的委托，那样事情更简单。

基尼奇: 好吧，我理解…那就先欠着。
基尼奇: 等你想好要什么，请随时和我说。

旅行者: 让您屈尊了。

### Talk 0 related talk 2

旅行者: 想一起聊聊天吗？
基尼奇: 你想打听什么？
旅行者: 不会内容不同，价格也不同吧？
基尼奇: 嗯，情报的价格由多方因素决定。
基尼奇: 但看你不紧不慢的态度，不像有急事…只是闲聊？
旅行者: 想听你聊聊猎龙的事。
基尼奇: 这不是有趣而轻松的话题。
阿乔: #啊哈，怎么了呀基尼奇？你是怕自己的冷酷吓到人家，还是怕残忍的细节让他睡不着觉呀？
旅行者: 我倒不在意。
基尼奇: 别听阿乔胡说，没那么可怕。
基尼奇: 龙是纳塔人的伙伴。但如果龙被深渊侵蚀到了极点，无法净化；或是受到他人蛊惑，无恶不作…
基尼奇: 这些情况下，为了避免灾害蔓延，只能人为终结它们的生命。
基尼奇: 绝境中，连丛林里的野兽都会奋力挣扎，何况凶恶的龙？
基尼奇: 因此，我一般先靠箭毒麻木龙的知觉，配合陷阱限制行动范围，最后趁对方虚弱时送它上路。
旅行者: 好像还是挺危险的…
基尼奇: 我不否认。但只要经验充足，准备齐全，「危险」也能被驯服。
基尼奇: 如果你对猎龙感兴趣，下次接到合适的委托，我来喊你。
基尼奇: 我们可以一起采摘有毒的浆果，合力布置陷阱…嗯，酬劳也会分给你的。
旅行者: 你遇到最难对付的龙是怎样的？
基尼奇: 是阿乔。
阿乔: 嗯嗯嗯？哪个不要命的二脚直立爬虫居然在当面议论我？
基尼奇: 贪婪与自负往往是恶龙的通病，容易利用。
基尼奇: 针对动作粗笨、空有力气的对象，则可以靠敏捷躲过前期攻势，寻找进攻机会。
基尼奇: 但阿乔不仅邪恶、强大，还保持着清醒的头脑，难能可贵。
阿乔: 哼，现在才想歌颂我来为自己捞好处？你算错了，我伟大圣龙可不吃这一套！
基尼奇: 收回我的话。你要是真的清醒，就会知道我不是在夸你。
阿乔: 别不好意思。都说了多少次，光是你们这些低劣生物的存在，就是对我此等高贵圣龙的衬托。我随时接受你们的奉承。
基尼奇: #唉…旅行者你看，是挺难对付吧？
旅行者: 呃…可以省略残酷的细节？

### Talk 0 related talk 4

旅行者: 有什么想做的吗？
基尼奇: 我无所谓。现在手边没委托，做什么都行。
基尼奇: 你想喝点东西吗？比如青蜜莓苦种果汁。

旅行者: 什么价格？
基尼奇: 呵，这杯不收你钱。

基尼奇: 后来，考虑到获取上的难度，我将它改成青蜜莓，相应调整了饮品中各材料的配比。
阿乔: 基尼奇！你故意的吧？平时怎么没见到你话那么多、解释起来那么啰嗦！
阿乔: 你要不要从维茨特兰的起源和世界上的第一棵火榴果树开始讲起啊？
基尼奇: 这是为了让我的朋友能对这道饮品有更多的了解，从而更好地享用它。
基尼奇: 反正你也尝不出果汁口感上的好坏，平时和你说你也不懂。
阿乔: 你…！
旅行者: 那个…我被你们说得想尝尝了！
基尼奇: 好。那我现在就为二位调配果汁。
旅行者: 那是什么？
基尼奇: 一种由我自创的混合燃素果汁。
