# Talk Group: NpcGroup - 2047

## Talk 0

旅行者: 你是？
乾子: 我是个矿工…准确地说，是前矿工，由于层岩巨渊被封锁了，因此目前是无业游民。
乾子: 虽然听说最近那边恢复了一部分生产，但复工名单里还没有我的名字。
乾子: 就是因为没事做，所以只能琢磨吃什么好，结果回过神都已经把附近所有的餐馆都吃遍了，正发愁今天吃什么呢。

### Talk 0 related talk 0

旅行者: 不工作没问题吗？
乾子: 这就不用您费心了！
乾子: 看你不像是千岩军的样子，我就透个底吧…
乾子: …实际上我在层岩巨渊做工时，偷偷把原料运出来过。这可是抢手货，像解翠行的石老板，一买都是一大筐！
乾子: 其实我也是想找个搭档…怎么样，有没有兴趣跟我一起干？
乾子: 从此以后吃香的、喝辣的，就算是搬到玉京台也是指日可待！
