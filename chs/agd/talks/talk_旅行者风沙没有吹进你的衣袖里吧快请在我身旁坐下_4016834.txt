# Talk Dialog

坎蒂丝: #旅行者，风沙没有吹进你的衣袖里吧？快请在我身旁坐下。
旅行者: 坎蒂丝是在…
坎蒂丝: 我在练习一首古老的曲子，我喜欢它曼妙的旋律，尽管它真正的结尾已在黄沙中佚失。
坎蒂丝: 过往我所做的，是重复演奏我能记住的段落。一遍，然后再来一遍。
坎蒂丝: 但今夜我正试图用我的理解，为它附上一个合适的结尾。
坎蒂丝: 或许是热闹的花神诞祭让心境发生了些许改变吧？我想有些故事适合被继续书写下去…
坎蒂丝: 有些记忆，即使模糊也不会改变其珍贵的本质。
旅行者: 这就回来了吗？
坎蒂丝: 听起来，我们的「花之骑士」仍有些意犹未尽。
坎蒂丝: 阿如村离不开我，而我也离不开它。
坎蒂丝: 我很荣幸能与大家共度一段愉快的节日时光，但回归生活本身并不是件需要落寞的事。
坎蒂丝: 打起精神，笑一笑吧。
坎蒂丝: 看看天边的云你就会明白，明天也一定会是个晴朗的好日子。
旅行者: 还有别的事…
坎蒂丝: 嗯，愿今日的欢笑一直留存在你我心中。
