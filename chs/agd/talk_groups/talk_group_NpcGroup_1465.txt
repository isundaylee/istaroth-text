### Talk 0 related talk 1

旅行者: 关于龙灾…
查尔斯: 没什么好说，这已经是过去的事情了。
查尔斯: 人生不只有甘甜的果汁，也需要苦涩的酒，这些事情也会成为我们经历的一部分。
查尔斯: 比起抱怨「为什么会发生」，不如继续向前看。学会接受苦涩的味道，这就是大人的做法。
查尔斯: 等你准备好接受这份苦涩了，我会给你来一杯特调的。

### Talk 0 related talk 2

旅行者: 关于龙灾…
查尔斯: 嗯…有些令人头疼，酒馆的进货渠道因为风魔龙而受到了一些影响，因此酒品的种类也变少了…
查尔斯: …至于风魔龙本身，我倒是不怎么担心，那些用酒精麻醉自己的客人纯粹是害怕过头了。
查尔斯: 退一万步讲，如果真的到了无法挽回的地步，那么只需要等待夜幕降临就好。
查尔斯: 那位「只属于夜晚的骑士」是不会坐视不管的。

### Talk 0 related talk 3

查尔斯: 有什么推荐吗？
查尔斯: 晨曦酒庄的精品苹果酒，可以说是「天使的馈赠」的主打酒品。
查尔斯: 感兴趣的话也可以尝尝我的调制酒，如果你是第一次喝，我可以送你一杯特调。
旅行者: 哪款烈酒比较好？
查尔斯: 喜欢烈酒的话，我推荐你尝试一下「午后之死」，这是凯亚队长的最爱。
查尔斯: 不过我见过最烈的酒，应该是那款名为「火水」的酒。
查尔斯: 这酒来自至冬，是至冬商人随使团来蒙德时带来的，不过迪卢克老爷认为不符合蒙德人的口味，就没有引进了。
查尔斯: 这仅存的一瓶是当初带来的样品，只作为展示不对外出售，要是真想尝尝的话就去至冬吧。
