# Talk Group: NpcGroup - 5257

## Talk 0

雷蒙多: 这个项目还算是有前景，这一个…唔，有待商榷…

### Talk 0 related talk 0

旅行者: 科学院的研究项目情况如何？
雷蒙多: 一些项目被重启启用，一些项目被暂时中止，一些研究员重新上岗，另一些则暂时休息…
雷蒙多: 按照舒瓦瑟尔先生的说法，一切都井井有条，按照着既有的秩序运行着，这就足够了，我可不想再接受一次「爆炸性惊喜」了。
雷蒙多: 秩序，是重中之重啊…

### Talk 0 related talk 1

旅行者: 你为什么认为是纳克偷走了核心？
雷蒙多: 您还记得，我在之前翻看了一下克洛妮艾小姐在废墟中拍摄的画片吗？
雷蒙多: 里面有一张纳克正在抑制器前操作的画片，虽然非常模糊…但我很确定，那个动作绝对不在操作规范里。
派蒙: 那也说明不了什么呀。说不定…说不定是有小偷从旁边的山头飞过来，然后把这个核心偷走了呢。
雷蒙多: 呵呵，这绝对不可能。这片废墟里的始基矿能量流就是最好的防盗措施，一般人可受不了这玩意。
雷蒙多: 而且，那台「实验性场力发生装置」的启动时机太巧了，绝不可能是刚好凑巧…
雷蒙多: 我有一个不太成熟的揣测。纳克…或者说，「窃取核心的人」应该早就卸除了「始基动能核心」，启动了备用能源。
雷蒙多: 但「芒荒湮灭能核心」是一种相对低效的能源，它必须要持续输出能量，才能达到这个大家伙的启动阈值。
雷蒙多: 所以，这人刻意活化了那颗大型始基矿，引起了一场不大不小，但颇能吸引注意力的爆炸。
雷蒙多: 这样，当院里派人上来的时候，一定会先维修始基矿抑制器。
雷蒙多: 当他们意识到，必须去拿「实验性场力发生装置」内的备用能源时，「实验性场力发生装置」也充能完毕。
雷蒙多: 从技术角度，这么做有极大的可行性。而在枫丹科学院内，能够做到这件事的，这样的人只有我、艾德温、还有纳克先生。
雷蒙多: 艾德温应该没有机会来做这件事…剩下两个嫌疑人，只有我和纳克先生了。
旅行者: 说不定就是你干的。
派蒙: 对呀，你之前说纳克撒谎，可你也一直在撒谎。
派蒙: 说不定就是自导自演，想骗我们保护你过来拿什么东西呢。
雷蒙多: 你们的指控不完全是没有道理。我之后也会申请一次院内的可信度调查。
雷蒙多: 但就我来看，犯下这次罪行的人，更可能是纳克先生。如果此事真是他所为，那在这次的事件中，他的获利最大。
雷蒙多: 就算我和他一起并列为嫌疑犯…那他被定罪的概率肯定也远大于我。

### Talk 0 related talk 2

旅行者: 你找到好项目了？
雷蒙多: 项目…呵呵，老实说，我并没有找到多少能够迅速展现其价值的项目。
雷蒙多: 在目前我所审核过的项目中，最好的也仅仅是「发展前景相对较为乐观」。
雷蒙多: 但这也没办法，毕竟所有有价值的项目，都已经被艾德温·伊斯丁豪斯先生亲手摧毁了。

### Talk 0 related talk 3

旅行者: 关于纳克…
雷蒙多: 院内已经对纳克先生的相关事宜展开了进一步调查，也派出了几名特别调查员，但目前尚没有找到有价值的线索。
雷蒙多: 哼…如果从个人的立场来说，我发自内心希望纳克先生和那枚「始基动能核心」一起灰飞烟灭。
雷蒙多: …但我也确切地意识到，枫丹科学院的运气不会这么好的。
