# Talk Group: NpcGroup - 1555

## Talk 0

旅行者: 她的故事确实很传奇。
艾迪丝博士: 是吧？她的一生，是抗争的一生，这应该就是蒙德的自由精神吧。
艾迪丝博士: 想要了解她的过去的话，不妨读一读这本书吧。

## Talk 1

旅行者: 关于大树…
艾迪丝博士: 这棵大树就是在当年温妮莎登上天空岛的地方生长出来的，我敢保证你走遍蒙德都找不到这么大的橡树！
艾迪丝博士: 关于它，我收集了数不胜数的故事和诗歌，找个地方坐坐吧，这些故事我跟你讲三天三夜也讲不完…
旅行者: …还是下次吧。
艾迪丝博士: 好吧，没关系。有时间的话随时欢迎！
旅行者: 天空岛？
艾迪丝博士: 天空岛是人间伟大的英雄最终相聚的地方。
艾迪丝博士: 登上天空岛而被接纳的英雄会成为诸神的一员，在高天之上继续守护人民。
艾迪丝博士: 凡人不会知道天空岛上是什么样的景色。
艾迪丝博士: 即使是行走大地的众神，也对天空岛缄口不言…
艾迪丝博士: 或许天空岛的故事，只有那些在人间故去的英雄才有资格讲述吧。

## Talk 2

艾迪丝博士: 你好，旅行者，要听我讲讲温妮莎的传奇故事吗？

### Talk 2 related talk 0

旅行者: 温妮莎？
艾迪丝博士: 是的，她就是解放蒙德的大英雄，自幼为奴，却推翻了贵族，为蒙德带来自由的勇士！
艾迪丝博士: 在传说中，她后来就是在这里登上了天空岛，从而化作了永远守护蒙德的神。
艾迪丝博士: 关于她的生平传说各异，但诸多故事却汇集在唯一的终点——就在这里。

### Talk 2 related talk 1

旅行者: 你是…？
艾迪丝博士: 啊啊，抱歉忘了自我介绍。
艾迪丝博士: 叫我艾迪丝就好，我是个历史作家和民俗研究者，最近在为我的作品取材。
艾迪丝博士: 你可能认识那个对丘丘人有着奇怪执着的学者，艾拉·马斯克，她是我的姐姐。

### Talk 2 related talk 2

旅行者: 再见。
艾迪丝博士: 再见，愿温妮莎注视着你！
