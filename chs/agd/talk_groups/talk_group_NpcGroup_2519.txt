## Talk 0

### Talk 0 related talk 0

旅行者: 对手？
可慎: 没错！对手有很多种——
可慎: 执行任务冲锋在前，率先与贼匪交战，是将贼匪当成对手，可以尽情施展浑身解数。
可慎: 演练时，与林教头真刀真枪比划，是将教头当成对手，演练与敌将对决…
可慎: 当然，教头的功夫胜我十年之上，通常都是我被教头打翻，教头点到即止。
可慎: 休息时，与镖师们切磋，也能好好练练拳脚功夫。
可慎: 长枪施展不开，被迫短兵相接时，这些武技肯定能派上用场。
可慎: 总之，每种对手都能锻炼我在不同环境下的战法。

可慎: 科目的广度几乎无可挑剔，可惜深度针对的是大部分普通将士。
可慎: 要是把这些科目练得滚瓜烂熟，还想提升实力，就得另辟蹊径了。
可慎: 我的人生目标是单独战胜玄文兽这样的魔兽，但我暂且还没得到岩王爷的认可，没有神之眼。
可慎: 所以，只能磨砺纯粹的武技，挑战躯体的极限。
