# Talk Dialog

旅行者: 在这里还习惯吗？
莱依拉: 你放心，这里的陈设完全能满足我的日常需要，一点可抱怨的都没有。
莱依拉: 一般来说，我很在意住的地方有没有小飞虫之类的。
莱依拉: 你知道的，就是那种四处乱飞，发出嗡嗡声，还会时不时骚扰你的飞虫。
旅行者: 是说飞萤吧？
莱依拉: 对对，就是那一类小虫子。
莱依拉: 别看它们小小的，但一到晚上夜深人静的时候，它们就会对着你的耳朵嗡嗡个不停，真的很烦人。
莱依拉: 我之前常被这种小虫子折腾得睡不着，但在你这里就完全没有这种困扰。
莱依拉: 又整洁又安静，真是失眠症患者的宝地！
旅行者: 是说萤火虫吧？
莱依拉: 不对啦，就算我的专业不是昆虫研究，也知道萤火虫和那种小虫子，肯定不是一个种类。
