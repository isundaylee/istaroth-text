## Talk 0

望涓: 有雅兴的鱼儿，请上钩…这一钩居然钓上了仪表不凡的异乡人？

### Talk 0 related talk 0

旅行者: 你为什么站在这里钓鱼？
望涓: 请大声点，我听不清~
望涓: 不过，看你的口型，应该和其他人一样，好奇我为什么在瀑布边钓鱼吧？
望涓: 请看那边的浅水区，有些不幸的鱼儿会从瀑布上坠落下来，撞在礁石上，一命呜呼。
望涓: 我为这些鱼儿的境遇感到遗憾，但也不能让它们白白死去。
望涓: 所以我会将它们烹饪成上好的菜肴，饱含感激地食用。
望涓: 多亏这些鱼儿，我每天都不用为食物发愁了，也就不需要钓取其他的鱼。
望涓: 放眼整个沉玉谷的话，每天死去的鱼儿因此减少了，这是一件善事。
望涓: 所以，现在我只会钓上那些有缘咬钩，或是有雅兴咬钩的鱼，权当和它们交个朋友了…
