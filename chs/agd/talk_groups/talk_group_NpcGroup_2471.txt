# Talk Group: NpcGroup - 2471

## Talk 0

阿绿: 嗯，潜力卓越的商家不可胜数，只缺一个品牌形象建设的指导者。

### Talk 0 related talk 0

旅行者: 你是？
阿绿: 你好呀，友善而热情的朋友。我是阿绿，本名小薇尔，来自枫丹廷的利奥奈区。
阿绿: 来到遗珑埠后，璃月的朋友们根据我名字的含义，给我起了个「阿绿」的称谓。
阿绿: 「绿」是茶叶的颜色，短期看来，能提升本地人对我的亲切感，有助于我打造个人品牌形象…
阿绿: 长期看，我会在遗珑埠长期停留，仔细研究并推广本地的美食。
阿绿: 嗯，我的第一本旅游指南，不如就叫《阿绿指南》好了！

### Talk 0 related talk 1

旅行者: 品牌形象？
阿绿: 怎么样？你也是第一次听说这个词吧？
阿绿: 本人小薇尔和姐姐小鲁热可是这个职业的代表人物哦！
阿绿: 虽然我们还没在枫丹廷打响名气，但有不少店家向我们咨询过方案！
阿绿: 我们会全方位帮助店家提升自己在公众视野中的形象，包括且不限于利用广告、新闻专栏、映影短片、大型活动等手段…
阿绿: 对于店家而言，他们的名气可能与实力不符，经过我们的宣传，他们便能获得与实力相符的人气。
阿绿: 根据我的情报，璃月的部分餐馆也存在这样的缺憾…
阿绿: 璃月的餐馆大多非常实惠，物美价廉，同价位中鲜有敌手。市民们只需少量摩拉就能享受丰盛的大餐。
阿绿: 不过，璃月也需要更多高端的餐饮品牌…毕竟厨师也需要「品牌形象」的支持。
阿绿: 如果所有餐馆都做小本生意，厨师被认为又累又赚不到多少钱，愿意学习厨艺的人可能就会变少。
阿绿: 为了厨师这个行业的未来，面向较为富裕的群体，赚取大量利润，给厨师开出高薪的餐馆，也是不可或缺的…
阿绿: 高低搭配，才能既让广大市民无忧无虑地享受美食，又让厨师成为可敬的职业，获得与技能相符的收入。
阿绿: 也许我能帮忙改善这一状况…嗯，前提是我对璃月的餐饮界有更深刻的认知。
阿绿: 调查还是得先从遗珑埠做起，慢慢推进啊…

### Talk 1 related talk 0

旅行者: 旅游指南？
阿绿: 嘿！我和我的姐姐小鲁热致力于发掘值得一去的景点，还有值得一试的菜肴。
阿绿: 美景和美食是全提瓦特的宝物，每个游客都应该抓住机会，好好享受一番！
阿绿: 但对不常造访某个地区的游客而言，从头询问当地人，慢慢了解该地的景点和菜肴…
阿绿: 未免也太麻烦了！可能要像我一样，花上好几年的功夫！
阿绿: 如果有一本…甚至是好几本足够细致全面的指南作为参考，那就省事得多！
阿绿: 正是因此，我和姐姐决定仔细探访提瓦特的每个聚居点，写作一系列旅游指南，帮助所有游客！
阿绿: 我们的计划实施得很顺利，中途只出了一点差错…
阿绿: 那就是，我姐姐发现她自己完全吃不惯枫丹之外的食物，只好和我交换职责…
阿绿: 以后，她的《阿红指南》就负责记录宝藏景点，我的《阿绿指南》负责宣传宝藏美食啦！
