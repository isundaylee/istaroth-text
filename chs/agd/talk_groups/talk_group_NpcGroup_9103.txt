## Talk 0

希诺宁: 你居然还有这种秘密基地，环境还这么好，不错啊。
希诺宁: 要是我也有一个这样的壶就好了。想偷懒了就往里面一钻，看谁还发现得了我。

### Talk 0 related talk 0

旅行者: 在这里还习惯吗？
希诺宁: 挺不错的。我还是第一次住在一盏壶里，和在部族里的感觉完全不一样，感觉很新奇。
旅行者: 希诺宁最喜欢哪个部族的环境？
希诺宁: 最喜欢？嗯…要是我爸在这里，听我说这话，肯定又要瞪我…
希诺宁: 我最喜欢的其实是玛拉妮那边的「流泉之众」啦。
希诺宁: 每天都可以泡温泉，空气也很湿润，有很多好喝的饮料，还有专门的首饰店和唱片店。
希诺宁: 无论从哪个角度来说，都很适合悠闲地生活啊…
希诺宁: 基尼奇那边的「悬木人」也不错，他们的帐篷都建得很高，只要躺在里面就能吹到很清爽的风。
希诺宁: 偶尔还能蹦个极啦，滑个草啦之类的…
旅行者: 不然，去其他部族里住一住？
希诺宁: 这个啊…哎呀，我其实试过啦。
希诺宁: 之前玛拉妮就邀请我去她家参加聚会，顺便留我在家里住了好几天。
希诺宁: 最开始我住得可舒服了，每天泡泡温泉，喝喝饮料，再和玛拉妮出去闲逛…
希诺宁: 但是…怎么说呢，越住我越…那个，怎么说呢…越住越不踏实。
希诺宁: 也不是住腻了，就是越来越想回「回声之子」去看看。
希诺宁: 到最后，我满脑子都是工坊里的事务，就匆匆向玛拉妮告别，回了部族里。
希诺宁: 也不知道为什么，一回到工坊…不，一进部族里，看见红色的峭壁时，我的心一下子就踏实了。
旅行者: 看来希诺宁还是最喜欢「回声之子」呢。
希诺宁: 是吗？我自己倒是没什么感觉…
希诺宁: 不过，我真的想再去「流泉之众」度个假啊…唉，过几天去问问玛拉妮吧。

### Talk 0 related talk 2

旅行者: 想一起聊聊天吗？
希诺宁: 行啊。我正好已经睡饱了，这盏壶里的温度正好，床和枕头也很舒服…
希诺宁: 感觉这一觉的质量够我回味一周的了。
旅行者: 那你怎么不多睡一会？
希诺宁: 差不多就可以了。我也不是那种极端嗜睡的人，只要保证每天有八九个小时的充足睡眠就足够。
希诺宁: 但是…在有工作上门的时候，我也很难一次性睡这么久。
希诺宁: 特别是打造「古名」，一旦开始，我每天都只能断断续续地睡三到四个小时，甚至连续好几天都不能睡觉。
旅行者: 打造「古名」很复杂吗？
希诺宁: 对我来说，打造「古名」在技术上并不复杂，但去求证那枚「古名」背后故事的真伪，就是一件天大的麻烦事了。
希诺宁: 一枚「古名」背后的故事，往往会横跨很长一段时间，牵涉多个相关人物。
希诺宁: 一般来说，这些相关人物都是在以自己的言行举止，践行对应「古名」之中所蕴含的「精神力量」…不然也不会被归到这枚「古名」之中。
希诺宁: 然后，问题就来了，这些相关人物早已作古，他们的一言一行，都只能通过卷宗来查证。
希诺宁: 但谁也不知道，这些卷宗里所记录着的人物言行到底是不是符合史实，所说的话究竟是不是准确无误。
希诺宁: 如果贸然把虚构的故事，和被讹传的话语锻造进「古名」里…
希诺宁: 那这枚「古名」一定会出现重大缺陷，甚至波及到持有这枚「古名」的人，让他的心神都出现问题。
旅行者: 但那么多故事，怎么查证真伪？
希诺宁: 一般我会去「烟谜主」那边，查阅一下他们的「织卷」。
希诺宁: 「织卷」上记录的历史一向非常可靠，而且…「烟谜主」那边还有茜特菈莉，我有时候也会去找她请教。
希诺宁: 如果「织卷」上也没有，那我就会去联系各个部族里的人，看能不能找到相关的资料。
希诺宁: 在搜集到大量的资料后，我会找出彼此之间有联系的故事，再对这些故事进行「交叉对比」。
希诺宁: 将查出来的谬误记录下来，再继续查找与之有关的故事，确定其究竟是谬误，还是漏传…
旅行者: 感觉是一个巨大的工程…
希诺宁: 是啊，所以我说，一旦开始打造「古名」，我就别想睡了。
希诺宁: 每天睁开眼睛是打造「古名」，闭上眼睛还是打造「古名」。
希诺宁: 玛拉妮曾经还自告奋勇地来帮我，结果…还没查证多少故事呢，她就沉沉地睡过去了，怎么叫都叫不醒。
希诺宁: 现在你知道，我为什么总是抽空去睡觉了吧？
希诺宁: 光是提起这些忙不完的工作，我就…呼啊…就有点犯困了…
旅行者: 以后我也来帮你！
希诺宁: 好啊，你旅行了那么久，肯定见多识广，也听过不少故事，应该有判断一段故事真伪的经验。
希诺宁: 下次打造「古名」的时候，我一定叫上你。
希诺宁: 对了，你来的时候记得带上个枕头，困了之后可以枕着睡觉。
希诺宁: 如果你和卡齐娜一样，枕着书睡觉的话，第二天醒来的时候，脖子会疼得直不起来。

### Talk 0 related talk 4

旅行者: 有什么想做的吗？
希诺宁: 没有。我不想做任何事，在这里发呆就挺好。
希诺宁: 难得在一个根本没人能找到我的地方休息，我就不给自己找麻烦了。
希诺宁: 来来，我们一起发发呆吧，我教你迅速进入发呆状态的诀窍。
旅行者: 这事情还有诀窍？
希诺宁: 当然了。发呆的本质，就是迅速清理自己脑子里乱七八糟的事情，再将自己的精神抽出身体，达到一种有专注性的完全平静…
旅行者: 有一种「烟谜主」祭祀的感觉…
希诺宁: 喔，我确实也向茜特菈莉学习过和冥想相关的知识。
希诺宁: 但我可没有…也懒得找那种和万物沟通的「灵性」，我只想学习到让自己清净下来的方法。
希诺宁: 好了好了，说起来复杂，其实很简单的，你跟着我一步一步来就行。
希诺宁: 首先，凝望着空气里的某一个点…随便哪里都可以，只要你觉得舒服。
希诺宁: 接着，开始放松你的眉头…再放松肩膀…专注盯着那个点，脑子里只思考，那个点究竟在哪里…
希诺宁: 然后…呼啊…然后…继续放松…放松…
希诺宁: …呼…呼…
旅行者: 发呆有这么深奥吗…
希诺宁: 当然啦。不信你去问茜特菈莉，或者欧洛伦，他们「烟谜主」里有许多人都要专门学习如何冥想。
希诺宁: 冥想和发呆其实差不多啦，都是先清理思想，再让自己的内心平静下来。
希诺宁: 两者的区别就在于，「烟谜主」需要在冥想状态进行某种行动，而我们，只需要享受发呆带来的平静就好。

希诺宁: 对我来说，打造「古名」在技术上并不复杂，但去求证那枚「古名」背后故事的真伪，就是一件天大的麻烦事了。
希诺宁: 一枚「古名」背后的故事，往往会横跨很长一段时间，牵涉多个相关人物。
希诺宁: 一般来说，这些相关人物都是在以自己的言行举止，践行对应「古名」之中所蕴含的「精神力量」…不然也不会被归到这枚「古名」之中。
希诺宁: 然后，问题就来了，这些相关人物早已作古，他们的一言一行，都只能通过卷宗来查证。
希诺宁: 但谁也不知道，这些卷宗里所记录着的人物言行到底是不是符合史实，所说的话究竟是不是准确无误。
希诺宁: 如果贸然把虚构的故事，和被讹传的话语锻造进「古名」里…
希诺宁: 那这枚「古名」一定会出现重大缺陷，甚至波及到持有这枚「古名」的人，让他的心神都出现问题。
旅行者: 但那么多故事，怎么查证真伪？
希诺宁: 一般我会去「烟谜主」那边，查阅一下他们的「织卷」。
希诺宁: 「织卷」上记录的历史一向非常可靠，而且…「烟谜主」那边还有茜特菈莉，我有时候也会去找她请教。
希诺宁: 如果「织卷」上也没有，那我就会去联系各个部族里的人，看能不能找到相关的资料。
希诺宁: 在搜集到大量的资料后，我会找出彼此之间有联系的故事，再对这些故事进行「交叉对比」。
希诺宁: 将查出来的谬误记录下来，再继续查找与之有关的故事，确定其究竟是谬误，还是漏传…
旅行者: 感觉是一个巨大的工程…
希诺宁: 是啊，所以我说，一旦开始打造「古名」，我就别想睡了。
希诺宁: 每天睁开眼睛是打造「古名」，闭上眼睛还是打造「古名」。
希诺宁: 玛拉妮曾经还自告奋勇地来帮我，结果…还没查证多少故事呢，她就沉沉地睡过去了，怎么叫都叫不醒。
希诺宁: 现在你知道，我为什么总是抽空去睡觉了吧？
希诺宁: 光是提起这些忙不完的工作，我就…呼啊…就有点犯困了…
旅行者: 以后我也来帮你！
希诺宁: 好啊，你旅行了那么久，肯定见多识广，也听过不少故事，应该有判断一段故事真伪的经验。
希诺宁: 下次打造「古名」的时候，我一定叫上你。
希诺宁: 对了，你来的时候记得带上个枕头，困了之后可以枕着睡觉。
希诺宁: 如果你和卡齐娜一样，枕着书睡觉的话，第二天醒来的时候，脖子会疼得直不起来。

## Talk 3

旅行者: 有什么想做的吗？
希诺宁: 没有。我不想做任何事，在这里发呆就挺好。
希诺宁: 难得在一个根本没人能找到我的地方休息，我就不给自己找麻烦了。
希诺宁: 来来，我们一起发发呆吧，我教你迅速进入发呆状态的诀窍。
旅行者: 这事情还有诀窍？
希诺宁: 当然了。发呆的本质，就是迅速清理自己脑子里乱七八糟的事情，再将自己的精神抽出身体，达到一种有专注性的完全平静…
旅行者: 有一种「烟谜主」祭祀的感觉…
希诺宁: 喔，我确实也向茜特菈莉学习过和冥想相关的知识。
希诺宁: 但我可没有…也懒得找那种和万物沟通的「灵性」，我只想学习到让自己清净下来的方法。
希诺宁: 好了好了，说起来复杂，其实很简单的，你跟着我一步一步来就行。
希诺宁: 首先，凝望着空气里的某一个点…随便哪里都可以，只要你觉得舒服。
希诺宁: 接着，开始放松你的眉头…再放松肩膀…专注盯着那个点，脑子里只思考，那个点究竟在哪里…
希诺宁: 然后…呼啊…然后…继续放松…放松…
希诺宁: …呼…呼…
旅行者: 发呆有这么深奥吗…
希诺宁: 当然啦。不信你去问茜特菈莉，或者欧洛伦，他们「烟谜主」里有许多人都要专门学习如何冥想。
希诺宁: 冥想和发呆其实差不多啦，都是先清理思想，再让自己的内心平静下来。
希诺宁: 两者的区别就在于，「烟谜主」需要在冥想状态进行某种行动，而我们，只需要享受发呆带来的平静就好。

## Talk 4

旅行者: 早啊，希诺宁。
希诺宁: 早上好，看你的样子，昨晚上睡得很不错啊。我的睡眠质量也很好。

## Talk 5

旅行者: 晚安，希诺宁。
希诺宁: 晚安，早点睡啊，睡之前就别看那些复杂的资料了，会睡不着或者做噩梦的。

## Talk 6

希诺宁: 最喜欢？嗯…要是我爸在这里，听我说这话，肯定又要瞪我…
希诺宁: 我最喜欢的其实是玛拉妮那边的「流泉之众」啦。
希诺宁: 每天都可以泡温泉，空气也很湿润，有很多好喝的饮料，还有专门的首饰店和唱片店。
希诺宁: 无论从哪个角度来说，都很适合悠闲地生活啊…
希诺宁: 基尼奇那边的「悬木人」也不错，他们的帐篷都建得很高，只要躺在里面就能吹到很清爽的风。
希诺宁: 偶尔还能蹦个极啦，滑个草啦之类的…
旅行者: 不然，去其他部族里住一住？
希诺宁: 这个啊…哎呀，我其实试过啦。
希诺宁: 之前玛拉妮就邀请我去她家参加聚会，顺便留我在家里住了好几天。
希诺宁: 最开始我住得可舒服了，每天泡泡温泉，喝喝饮料，再和玛拉妮出去闲逛…
希诺宁: 但是…怎么说呢，越住我越…那个，怎么说呢…越住越不踏实。
希诺宁: 也不是住腻了，就是越来越想回「回声之子」去看看。
希诺宁: 到最后，我满脑子都是工坊里的事务，就匆匆向玛拉妮告别，回了部族里。
希诺宁: 也不知道为什么，一回到工坊…不，一进部族里，看见红色的峭壁时，我的心一下子就踏实了。
旅行者: 看来希诺宁还是最喜欢「回声之子」呢。
希诺宁: 是吗？我自己倒是没什么感觉…
希诺宁: 不过，我真的想再去「流泉之众」度个假啊…唉，过几天去问问玛拉妮吧。
