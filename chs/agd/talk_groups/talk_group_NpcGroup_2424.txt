## Talk 0

楚婆婆: 啊…是旅行者吗。
楚婆婆: 我刚在专心听「他」讲话，没注意到你进来。

### Talk 0 related talk 0

旅行者: 我没听到其他人讲话…
楚婆婆: 呵呵…我一开始的时候也是听不到的。
楚婆婆: 可是几十年过去了，我的耳朵也越来越不灵了，但最近却开始能够听到「他」了…
楚婆婆: 那个传说是真的，太好了，不枉我守在这里这么多年。
旅行者: 「他」是谁？
楚婆婆: 是我的丈夫，他有着古铜色的皮肤，健硕的肌肉…
楚婆婆: 在求婚时候却像个孩子一样，郑重其事地说要和我白头偕老，呵呵。
楚婆婆: 我们是在同一个渔村长大的，做的也是出海打渔的活计。
楚婆婆: 直到我们结婚的第四年，风暴肆虐的海上，他把我绑在破损的舢板上…
楚婆婆: 我漂泊了一整夜，漂到这里捡回了自己的命，却再没有机会兑现和他白头偕老的誓言…
旅行者: 传说指的是…
楚婆婆: 闭上眼睛，仔细对着海螺聆听，就可以听到来自大海的声音…你听说过吗？
楚婆婆: 那么我那位永远留在了大海中的丈夫…会不会为我留下了什么话语呢…
楚婆婆: 所以我搬到了这座海螺屋里来住，一直等待着…
楚婆婆: 是否有一天，能听到海浪声中夹挟着的他的声音。

### Talk 0 related talk 1

旅行者: 打扰了。
楚婆婆: 出海要当心，再见。
