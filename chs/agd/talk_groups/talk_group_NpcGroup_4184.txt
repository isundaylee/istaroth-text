# Talk Group: NpcGroup - 4184

## Talk 0

兰拉迦: #森林中的一切都不会被遗忘。那菈旅行者，你的故事会永远在兰那罗之中流传下去吧。

### Talk 0 related talk 0

旅行者: 关于「无留陀」…
兰拉迦: 那是很久以前的故事了…
兰拉迦: 在过去，兰那罗行走在大地上，和那菈及万物为友，而林中也没有无留陀。直到在很多很多个月亮以前的一次灾难…
兰拉迦: 树王归还了沙恒，无留陀开始在地上肆虐。而丛林终于恢复生机后，兰那罗为了保护自己，将真正的桓那兰那藏在了梦中。
兰拉迦: 森林本不知死为何物。直到有一天，树想起大地中充满了凋败的花、死去的兽。
兰拉迦: 正如在同一天，大地想起自己曾经或将要吞食漆黑的毒血。河流想到太阳降下，蒸发了所有水的红色的世界。
兰拉迦: 他们明白了，万物都将无可避免地死去。这时，「无留陀」就诞生了。此乃「死」之名，知晓它存在的，终究会死去。
兰拉迦: 无留陀的力量就是令一切走向死。而高大、阴暗、扭曲的东西就是无留陀选中的化身，带来死的使徒。
兰拉迦: …如今，你和兰罗摩击败了无留陀的化身，而且森林中有了新的觉王树，一切都会越来越美好吧。

### Talk 0 related talk 2

旅行者: 关于「森林书」…
兰拉迦: 那是那菈法留纳留下的东西，兰罗摩将它交给你，老身觉得他做得很对。
兰拉迦: 其中已经充满了你与兰那罗们一起的故事吧。老身知道，那菈长大以后，总会忘记如何做梦，忘记我们。
兰拉迦: 但是，只要「森林书」中还写着我们的事情、我们的名字，就算你将来失去了关于我们的记忆，也可以当成故事看。

### Talk 0 related talk 3

旅行者: 关于兰罗摩…
兰拉迦: 老身十分想念他。他成为了新的觉王树之后，老身的梦中也少了一分喧闹。
兰拉迦: 那菈与兰那罗一同玩闹，一同行走在树荫下。老身如今已经不记得那样的时代了，但一直记得这样的故事。
兰拉迦: 老身原本期望，兰罗摩还是种子的时候，这样的时候能够到来。不过，一切都会回到沙恒当中，沙恒中的那菈与兰那罗不会有任何隔阂。
兰拉迦: #而且，老身也非常高兴，兰罗摩能与那菈旅行者成为朋友。

### Talk 0 related talk 4

旅行者: 关于兰拉娜…
兰拉迦: 兰拉娜也是十分古老的兰那罗，经历过许多花开、许多雨露。
兰拉迦: 因为太过喜爱那菈、太过喜爱森林，所以一直不愿成为莎兰树。如果成了莎兰树，那就无法与那菈一同玩耍了。
兰拉迦: 老身听说，兰拉娜与那菈拉娜一同去旅行了。希望旅行中有许许多多的故事，有许许多多美好的事物，能填补兰拉娜所失去的。
