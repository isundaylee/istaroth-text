# Talk Group: NpcGroup - 2006

## Talk 0

狗三儿: 说你呢，鬼鬼祟祟的，干什么呢？

### Talk 0 related talk 0

旅行者: 我就不能是打工的人吗？
狗三儿: 什么？来接差事的工人？那你有点不走运…
狗三儿: 能在这间仓库接到的工作，除了看门和货物的保养，主要就是搬货了。
狗三儿: 在码头和仓库之间来回搬货可不轻松，一般来说，白天搬货的单子都给了常年在码头正式工作的弟兄们。
狗三儿: 只有夜晚的差事才会分给有空打零工的人，要是处理不完，再留给其他弟兄。
狗三儿: 但最近来了个又高又壮，脑袋上戴着红色尖角的大兄弟，干活特别勤快。
狗三儿: 那个大兄弟好像根本不会累，一人干了差不多三个人的活，每晚只做三小时的差事，就能赚到不少摩拉。
狗三儿: 他倒也挺通情达理，和我们好好解释了一番，说是最近急着用钱，只会这样干几天，不会长期抢别人的单子。
狗三儿: 所以你最近深夜来做工，也接不到什么活了。

### Talk 0 related talk 1

旅行者: 只是路过。
狗三儿: 哦，路过啊…
狗三儿: 我告诉你，你别打这仓库的主意，我可是一直看着你呢。

### Talk 0 related talk 2

旅行者: 这座仓库很重要吗？
狗三儿: 看见码头上的船了吗？不论大小，不论开进开出，船上货物饮水都要这座仓库经手。
狗三儿: 有我在这儿，璃月的仓库就是全世界最安全的仓库！
狗三儿: 我们建造仓库的岩石，风吹不动，火烧不进，绝对安全！你是旅行者，你见的世面多，你到过须弥吗？
狗三儿: 我听说那个国家的仓库是用草搭的棚子…你说那能安全吗？有四个我守着的话，那还差不多…

### Talk 0 related talk 3

旅行者: 不好意思。
狗三儿: 没打什么主意就行…你这么说我都不好意思了。
