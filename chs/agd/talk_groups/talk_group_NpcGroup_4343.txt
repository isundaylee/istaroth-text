### Talk 0 related talk 0

旅行者: 雇佣兵是？
伊萨米: 啊，我说的是那位在附近看守的沙欣先生。
伊萨米: 像他这么爱问问题的镀金旅团成员，我还是第一次见到。
伊萨米: 我曾以为镀金旅团里尽是些寡言少语的硬派家伙，真是大错特错了。
伊萨米: 我不讨厌热情的人，但他总是问这里是做什么的、那里都放着些什么东西…让我有些吃不消。
伊萨米: 主要是，看他那个好奇的样子，确实不像是图谋不轨。我也不至于因为他口头的几个问题，就劳动风纪官出马…
伊萨米: 所以，也只好尽量敷衍他了。

### Talk 0 related talk 1

旅行者: 这里是？
伊萨米: 这里名为禅那园，是隶属于教令院生论派的研究场所。你在这里可以见到研究植物、蕈类，甚至昆虫等课题的人员。
伊萨米: 大部分都是来自教令院的学者或学生，也有一些是与禅那园有着紧密关联的合作者。
伊萨米: 比如化城郭有一位巡林官，他以前上学时就整天泡在这里，现在即便离开了教令院，也经常回来做研究。
伊萨米: 他的研究成果都很宝贵，所以禅那园这边也愿意为巡林员们的工作提供协助。算是一种互惠互利的良好合作吧。
