## Talk 0

老刘伯: 哦，年轻人，找老朽有什么事吗？

### Talk 0 related talk 0

旅行者: 您在做什么？
老刘伯: 老朽只是在回想些过去的事，等你到了我这个年纪就会发现，比起展望未来还是过去的回忆更美好一些。
老刘伯: 就比如我最近常常想起，住在隔壁的姐姐淑之年轻时候的样子…
老刘伯: 我总趴在窗边偷看她，有时候她发现我了，还会对我微微一笑。
老刘伯: 我永远都不会忘记她笑起来时脸颊上那对深深酒窝，饱含笑意的眼眸中仿佛只剩下了我…
旅行者: 后来呢？
老刘伯: 后来？什么后来？
老刘伯: …难道你是指告、告白？
老刘伯: 你、你在说什么呢，告白那种事，哪儿有你说的那么简单啊！
老刘伯: 嘿呀，现在的年轻人啊…羞耻心都不知道去哪儿了…

旅行者: 再见。
老刘伯: 有道是「人无再少年」，到头来别像我一样，除了回忆什么也不剩下…
