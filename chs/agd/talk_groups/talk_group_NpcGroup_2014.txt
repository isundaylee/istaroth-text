# Talk Group: NpcGroup - 2014

## Talk 0

慕胜: 外来人吗？你也上柱香吧？

### Talk 0 related talk 0

旅行者: 关于岩王帝君的死…
慕胜: 像岩王爷这样的神仙怎么会死呢？就算是我亲眼看到了，我也不相信。
慕胜: 岩王爷只是搬到那天空岛上去住啦，那儿应该有不少他的仙人老伙计。
慕胜: 至于璃月，岩王爷也一定会安排得妥妥当当，要问为啥…
慕胜: 就比方说，我在老家住了几十年，出来上香这几天还时不时会想家呢…
慕胜: 更何况岩王爷，可是在璃月住了千年啊…肯定放不下我们的。

### Talk 0 related talk 1

旅行者: 你是？
慕胜: 我是个乡下人，每年只有这段时间才会进城，来给岩王爷上柱香。
慕胜: 璃月的百姓都喜欢讨个彩头，图个吉利，我也一样。
慕胜: 如果岩王爷不灵，如今的璃月哪里会这么富裕？所以不管路再远，事情再多，我每年都会来这倚岩殿。
慕胜: 也就是尽人事，而知天命吧。
旅行者: 关于岩王爷的保佑…
慕胜: 怎么，听你的口气，你是不信吗？
慕胜: 就在前段时间，在来璃月港的路上我遇到了劫匪…幸亏有一位自称「古华派行秋」的少侠出手相助，我这才平安无事。
慕胜: 这一定是岩王爷的护佑！他老人家肯定知道我是来上香的，所以我才会获救！
慕胜: …要不是岩王爷的安排，古华派这种消失了多年的门派怎么会突然冒出来啊？
慕胜: 不管怎么说，感谢岩王爷。我也准备顺便帮古华派的那位少侠上柱香，为他祈福平安，就当做我的报答。
