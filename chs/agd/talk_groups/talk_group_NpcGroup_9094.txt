# Talk Group: NpcGroup - 9094

## Talk 0

千织: #旅行者，你来了啊，还以为你平时不常回家。
千织: 旅行如何？有什么精彩的事发生了吗？说不定能为我提供灵感。

### Talk 0 related talk 0

旅行者: 在这里还习惯吗？
千织: 挺不错，安静惬意，风景怡人。是个疗愈身心的好地方。
千织: 在枫丹廷待腻了，偶尔就想出来旅游一趟。没有不守规矩的顾客，也没有胡说八道的报纸。
旅行者: 千织去过什么地方旅游？
千织: 那多了去了。当时从稻妻到枫丹，可不是几天就能走完的路程。
千织: 唱诗饮酒的蒙德城、商业繁荣的璃月港，还有须弥热闹的大巴扎和怎么都望不到头的沙漠…
千织: 我当时喜欢到最热闹的地方，找个椅子，就这么坐一天。观察人们的衣装、感受那里的氛围、猜想他们的故事…
千织: 有时我会觉得，现在做衣服的很多灵感，就是那时在旅途中储存下来的。你应该能理解这种感受。
千织: 说了半天，还是绕回了我的工作上。
千织: 唔…恐怕在生活中汲取灵感，已经成为我的习惯了。
千织: 总之，我喜欢旅游。说不定以后有机会的话，也会在旅途中与你相见。
旅行者: 千织想去什么地方旅游？
千织: 去之前没去过的地方。这样才能酝酿出新的灵感。
千织: 这么说的话，纳塔和至冬我都没去过…是不是该计划计划了。
千织: 每个国家的服饰文化都有很大区别，不断吸收与接纳、融合与创新…这才是「千织屋」能引领时尚的秘诀。

### Talk 0 related talk 2

旅行者: 想一起聊聊天吗？
千织: 哟，真是稀奇。很多人觉得我说话不好听，都不会主动来找我聊天。
千织: 我其实只是不喜欢绕弯子而已，有这点相互顾及的时间，正事早就做完了。
千织: 你呢？你喜欢哪种？
旅行者: 我也喜欢直来直去。
千织: 真好，这样能省去很多不必要的麻烦。
千织: 我想起来有一次，一位客人在米色和灰色的布料间踌躇不定。
千织: 我说米色适合你，能凸显身材，和你的发色也很搭。结果她嘟着嘴，说觉得灰色更耐脏。
千织: 我说没问题，灰色的话我帮你调整一下版型，会显得更加大气。结果她又嘟着嘴，说灰色恐怕太沉闷了。
千织: 这么一来二去，一下午就过去了。最后她选了款红色的…
千织: 我倒不是不能理解，但要我说的话，这些人思考的时候都缺少了「最重要的是什么」这个前提。
千织: 最在意的是身形就选米色、最在意的是耐用就选灰色…
千织: 但如果你最在意的，是暗恋的人在宴会上能一眼找到你，那拜托进店门的时候就告诉我这点！
千织: 呼…光是想想都来气，让我缓缓。
旅行者: 可能那人害羞…
千织: 你说得对，她有所顾虑。但明明直接说出来的话，能省下更多时间，也能让我更好地依照她的需求进行定制。
千织: 这种事见得多了，我就越发告诫自己说话不要藏着掖着。
千织: 这或许不是最聪明的做法，但对我来说，是效率最高的做法。
千织: 所以我不会对你隐瞒自己的感受与想法，希望你也如此。
千织: 当然，你要是对我说「千织屋的衣服真难看」，那记得附上你的理由和见解，我会作为参考。
千织: 单纯的「无理取闹」，我想你是做不出来的。除非你好奇地板的触感。
旅行者: 还是委婉一点好…
千织: 是吗？但这对我来说还挺困难。
千织: 可能我从没有过模棱两可的时候吧，要么就是喜欢，要么就是不喜欢。

千织: 前提是你能赢过我。
旅行者: …是不是有些不公平？

千织: 要是你输了，就来我店里打几天工吧。打扫店铺、整理衣服，偶尔当当模特…
千织: 要是我输了，就随你开一个差不多的条件，打扫你的家，或者做点好看的东西帮你装点一下什么的…

千织: 如此纷繁夺目，络绎不绝的街道上，有着「千织」这个时尚王国的第一块领土。
千织: 我会继续延展它的版图。在此期间，也随时欢迎你来做客。

## Talk 8

旅行者: 在这里还习惯吗？
千织: 挺不错，安静惬意，风景怡人。是个疗愈身心的好地方。
千织: 在枫丹廷待腻了，偶尔就想出来旅游一趟。没有不守规矩的顾客，也没有胡说八道的报纸。
旅行者: 千织去过什么地方旅游？
千织: 那多了去了。当时从稻妻到枫丹，可不是几天就能走完的路程。
千织: 唱诗饮酒的蒙德城、商业繁荣的璃月港，还有须弥热闹的大巴扎和怎么都望不到头的沙漠…
千织: 我当时喜欢到最热闹的地方，找个椅子，就这么坐一天。观察人们的衣装、感受那里的氛围、猜想他们的故事…
千织: 有时我会觉得，现在做衣服的很多灵感，就是那时在旅途中储存下来的。你应该能理解这种感受。
千织: 说了半天，还是绕回了我的工作上。
千织: 唔…恐怕在生活中汲取灵感，已经成为我的习惯了。
千织: 总之，我喜欢旅游。说不定以后有机会的话，也会在旅途中与你相见。
旅行者: 千织想去什么地方旅游？
千织: 去之前没去过的地方。这样才能酝酿出新的灵感。
千织: 这么说的话，纳塔和至冬我都没去过…是不是该计划计划了。
千织: 每个国家的服饰文化都有很大区别，不断吸收与接纳、融合与创新…这才是「千织屋」能引领时尚的秘诀。

## Talk 9

旅行者: 想一起聊聊天吗？
千织: 哟，真是稀奇。很多人觉得我说话不好听，都不会主动来找我聊天。
千织: 我其实只是不喜欢绕弯子而已，有这点相互顾及的时间，正事早就做完了。
千织: 你呢？你喜欢哪种？
旅行者: 我也喜欢直来直去。
千织: 真好，这样能省去很多不必要的麻烦。
千织: 我想起来有一次，一位客人在米色和灰色的布料间踌躇不定。
千织: 我说米色适合你，能凸显身材，和你的发色也很搭。结果她嘟着嘴，说觉得灰色更耐脏。
千织: 我说没问题，灰色的话我帮你调整一下版型，会显得更加大气。结果她又嘟着嘴，说灰色恐怕太沉闷了。
千织: 这么一来二去，一下午就过去了。最后她选了款红色的…
千织: 我倒不是不能理解，但要我说的话，这些人思考的时候都缺少了「最重要的是什么」这个前提。
千织: 最在意的是身形就选米色、最在意的是耐用就选灰色…
千织: 但如果你最在意的，是暗恋的人在宴会上能一眼找到你，那拜托进店门的时候就告诉我这点！
千织: 呼…光是想想都来气，让我缓缓。
旅行者: 可能那人害羞…
千织: 你说得对，她有所顾虑。但明明直接说出来的话，能省下更多时间，也能让我更好地依照她的需求进行定制。
千织: 这种事见得多了，我就越发告诫自己说话不要藏着掖着。
千织: 这或许不是最聪明的做法，但对我来说，是效率最高的做法。
千织: 所以我不会对你隐瞒自己的感受与想法，希望你也如此。
千织: 当然，你要是对我说「千织屋的衣服真难看」，那记得附上你的理由和见解，我会作为参考。
千织: 单纯的「无理取闹」，我想你是做不出来的。除非你好奇地板的触感。
旅行者: 还是委婉一点好…
千织: 是吗？但这对我来说还挺困难。
千织: 可能我从没有过模棱两可的时候吧，要么就是喜欢，要么就是不喜欢。

## Talk 10

旅行者: 有什么想做的吗？
千织: 你这儿有羽子板吗？想和你来打几场。
千织: 你脸上的表情是怎么回事？该不会在你心中，我只会坐在工作台前缝衣服吧？
旅行者: 只是没想到是羽子板…
千织: 小时候在稻妻常玩，当时可没几个孩子能赢我。
千织: 结果到了枫丹之后没人会玩，教新手打又觉得像在欺负人。
千织: 你在稻妻的时候应该玩过这种游艺吧？想必身手不赖。
千织: 怎么？难不成你怕了？
旅行者: 接受挑战！
千织: 嗯，有气势。不愧是人们交口称赞的旅行者。
千织: 先说好，要是输了，可别指望我帮你保密。
千织: 准备好就开始，我用自己的两支羽子板。
旅行者: …二板流？
千织: 规则没说不能用两支呀。我平时用双刀，现在用两支羽子板也很正常吧？
千织: 你要是想的话，用四支来和我打都行。我没意见。
千织: 还是说，这就已经准备认输了？
旅行者: …输了别不认！
千织: 放心，虽然「千织」这个品牌的理念是永不认输。但对朋友，我可不会耍赖。
千织: 前提是你能赢过我。
旅行者: …是不是有些不公平？
旅行者: 输了的人怎么办？
千织: 哦？来认真的？好呀，有点代价才好玩。
千织: 要是你输了，就来我店里打几天工吧。打扫店铺、整理衣服，偶尔当当模特…
千织: 要是我输了，就随你开一个差不多的条件，打扫你的家，或者做点好看的东西帮你装点一下什么的…
