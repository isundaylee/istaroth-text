# Talk Group: NpcGroup - 30252

## Talk 0

茅葺一庆: 嗨，怎么了吗？
旅行者: 关于「百人一揆」…
茅葺一庆: 「百人一揆」最初是由「天狗党人」举办的庆典仪式，旨在宣扬打闹的好处。
茅葺一庆: 然而，在「天狗党人」被取缔后，还是会不时有人秘密举办「百人一揆」…虽然没人知道举办者的身份，但那也不重要，不是吗？
茅葺一庆: 总之，现在的「百人一揆」是相当热闹的乱斗大会，不管是武者、还是魔物，都能自由参加这场令人热血沸腾的祭典。
茅葺一庆: 对于渴求争斗的人来说，再合适不过了。
旅行者: 关于「天狗党人」…
茅葺一庆: 「天狗党人」是过去活跃在稻妻的团体，他们崇尚天狗一族的生活方式，将无所畏忌的战斗视为自我实现的途径。
茅葺一庆: 无论是心情低落的时候，还是有值得庆贺的喜事发生的时候，甚至是单纯感到无聊的时候，都可以展开酣畅淋漓的争斗…
茅葺一庆: …所以，后来，「天狗党」就因为扰乱公序良俗、破坏社会治安的缘故，被天领奉行取缔了。
旅行者: …真的被取缔了吗？
茅葺一庆: 嗯？不知道，我从来没有想过这个问题…
茅葺一庆: 不过…应该是很早之前就被取缔了吧，我已经很久没听过他们的传闻了。
旅行者: …这东西不会是你举办的吧？
茅葺一庆: 不不不，怎么可能嘛！
茅葺一庆: 咳…虽然我确实很期待看到大家战斗的英姿，但只凭我一个人，也没钱给胜利者们发奖品嘛…
旅行者: 关于你的小说…
茅葺一庆: 这个…说起来有些遗憾，之前的容彩祭上，《吹向金发武士的死亡之风》没怎么卖出去。
茅葺一庆: 不过没关系，那只是《异邦人喧哗录》系列的序章，下一部作品一定会广受好评的。标题就叫《花之夺命剑》！
茅葺一庆: 虽然我还没想好，具体的内容应该怎么写…但我相信，看过你这次的战斗，我一定能写出最棒的作品！
旅行者: 你还是不参赛吗？
茅葺一庆: 我吗？我实在是没有战斗的才能，只希望自己能够将武者们耀眼的样子记录下来。
茅葺一庆: 尤其是在目睹了你的剑技后，我觉得，比起直接参赛，用文字写下那如同天光一般隽永的剑之舞，才是我的使命所在。
茅葺一庆: 若是能在字里行间重现如此绚烂的光彩，应该也能称得上是「武道」的极致吧？
旅行者: 没什么。
茅葺一庆: 那么，这一次，也请让我见证你闪耀的身姿吧！
