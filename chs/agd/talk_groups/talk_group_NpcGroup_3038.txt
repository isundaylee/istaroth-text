## Talk 0

蓝川丞: 「秋沙钱汤」，温暖心房，带您体验，纳塔风光。亦可诊疗，心中忧烦。
蓝川丞: 哦呀，竟然是异国的旅行者，稀客啊，欢迎光临！正好本店的大股东也回来了，你们应该有不少共同话题！

### Talk 0 related talk 0

旅行者: 纳塔风光？
蓝川丞: 哈哈，说风光可能夸张了些…「秋沙钱汤」呢，是我很多年前在纳塔旅行，体验过火山温泉之后，回稻妻开的温泉馆。
蓝川丞: 火山温泉的治愈，真是泡过一次就永生难忘。我旅归家乡后，就总想着也要稻妻的亲朋好友们体会一次那种温暖、舒缓的感觉。
蓝川丞: 没想到「秋沙钱汤」慢慢建起来后，越来越受欢迎，就连大人物们也经常光临，搞得我常常后悔自己定价定低了呢，哈哈。

### Talk 0 related talk 1

旅行者: 大股东？
蓝川丞: 嗯，我在纳塔得到灵感，打算回稻妻开温泉馆时，资金并不充裕，于是四处寻找投资人。
蓝川丞: 结果，有位在「流泉之众」旅居的心理诊疗师，也就是可敬的梦见月瑞希女士，听说了这件事。
蓝川丞: 她也非常赞成在稻妻开设温泉馆，于是给这个项目投了一大笔钱。
蓝川丞: 虽然以前，稻妻城的街坊邻居们总是叫我老板，但要论「秋沙钱汤」真正意义上的话事人，还得是梦见月老板。
蓝川丞: 我最多只是个管商业运营的经理，「秋沙钱汤」的经营重点和发展方向，都由梦见月老板说了算。
蓝川丞: 梦见月老板归国后，就在温泉馆原有的经营基础上做了相当程度的改进，取得了惊人的成效…
蓝川丞: 假如你被不安的情绪困扰，不妨来找梦见月老板聊一聊，肯定会有不错的收获！
