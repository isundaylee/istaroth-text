# Talk Group: NpcGroup - 1581

## Talk 0

旅行者: 你的家族…
芬德: 多数人平静的生活，往往需要少数人付出牺牲，而我的家族恰好正是这少数人的一份子。
芬德: 不过我的家族在得到温妮莎大人的命令时，便已经有了觉悟。
芬德: …你听说过「风神杂烩菜」吗？传闻中是温妮莎大人击败旧贵族后，庆功宴上的一道名菜。
芬德: 我的祖先也有幸与温妮莎大人共享盛宴，事后还根据味道请大厨推断出了菜谱。
芬德: 那味道如同清风拂过耳畔，胸中仿佛回荡着风神对于安宁与自由的祝愿…正是这道菜支撑着我们一族度过千百个守望风暴的日日夜夜。
芬德: …感谢你的聆听，我想把这种感觉也传达给你，菜谱就请你收下吧，不必客气。

## Talk 1

旅行者: 过去的事情？
芬德: 正如你所见。从瞭望塔上看去，现在蒙德的海上一片风平浪静。
芬德: 自从「那个时代」过去以后，巴巴托斯大人降下了守护，蒙德便再没受过风暴的侵扰。
芬德: 即使是我们家族，也已经好几代人没有见过真正的风暴了…

## Talk 2

旅行者: 但你依然在守望呢。
芬德: 既然是「烽火」，都会渴望有燃烧的一天吧。
芬德: 哪怕我知道风暴再也不会出现，还是会偶尔回忆起父辈曾经讲述过的…
芬德: 家族先祖们在狂风暴雨中吹响号角，为整个蒙德预警的身姿…
芬德: 回过神来的时候，自己不知不觉地就又走到这里来了。

## Talk 3
