## Talk 0

### Talk 0 related talk 0

旅行者: 关于智慧宫…
卡塔扬: 智慧宫是教令院的藏书阁，学者或学生可以在这里借阅部分通识书籍。
卡塔扬: 还有不少藏书仅对高级学者开放，作为文献凭证使用。
卡塔扬: 毕竟大部分的知识都能通过虚空获取，很多人阅读书籍仅仅是想亲眼见证知识的来源罢了。
卡塔扬: 所以虽然大家都说智慧宫就是须弥的图书馆，但我倒觉得这里更像是博物馆呢。

### Talk 0 related talk 2

旅行者: 关于你自己…
卡塔扬: 我是这里的图书管理员，负责处理借阅申请和书籍整理等工作。
卡塔扬: 我家世世代代都在教令院担任与书相关的职务，我从小就喜欢新书散发的油墨味道，图书管理员是我梦寐以求的工作呢。
卡塔扬: 不过在入职了之后我才发现，这份工作比我想象的要简单许多。

卡塔扬: 所以需要我处理的事情很少，基本就是整理整理书架而已。
